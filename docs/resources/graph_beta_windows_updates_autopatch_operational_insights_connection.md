# microsoft365_graph_beta_windows_updates_autopatch_operational_insights_connection (Resource)

Manages a Windows Update Operational Insights connection using the `/admin/windows/updates/resourceConnections` endpoint. An Operational Insights connection links Windows Update for Business deployment service to an Azure Log Analytics workspace, enabling deployment reporting and insights. This resource does not support in-place updates — any change to a field will destroy and recreate the connection.

## Microsoft Graph API Permissions

The following client `application` permissions are needed in order to use this resource:

**Required:**
- `WindowsUpdates.ReadWrite.All`

**Optional:**
- `None` `[N/A]`

## Dependency Notes

The Azure Log Analytics workspace referenced by `azure_resource_group_name`, `azure_subscription_id`, and `workspace_name` must already exist before creating this resource. The Microsoft 365 tenant's service principal must be granted access to the workspace.

## Field Mutability

All fields on this resource require replacement — no in-place update is supported by the API. Any change to any attribute will destroy and recreate the connection.

| Field | Mutable after creation |
|-------|------------------------|
| `azure_resource_group_name` | No — changes require resource replacement |
| `azure_subscription_id` | No — changes require resource replacement |
| `workspace_name` | No — changes require resource replacement |
| `state` | Read-only (computed) |
