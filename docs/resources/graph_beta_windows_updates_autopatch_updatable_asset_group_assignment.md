# microsoft365_graph_beta_windows_updates_autopatch_updatable_asset_group_assignment (Resource)

Manages membership of a Windows Autopatch updatable asset group using the `/admin/windows/updates/updatableAssets/{groupId}/addMembersById` and `removeMembersById` endpoints. This resource manages which Entra ID devices are members of a given updatable asset group. When the resource is created, devices are added to the group. When updated, the diff between plan and state is applied. When deleted, all managed devices are removed from the group. The updatable asset group must already exist, managed by the `microsoft365_graph_beta_windows_updates_autopatch_updatable_asset_group` resource.

## Dependency Notes

This resource requires an existing `microsoft365_graph_beta_windows_updates_autopatch_updatable_asset_group` resource. The `updatable_asset_group_id` attribute must reference the ID of that group.
